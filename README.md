# ReactJs CRUD



## Prerequisite
* Nodejs v6+
* yarnpkg (optional)


## Getting Started
```sh
git clone https://github.com/kentitop/reactCRUD.git

cd reactCRUD

yarn install
    or
npm install

yarn start
    or
npm start
```

Open [http://localhost:8080](http://localhost:8080)<br>


## Libraries used
* ReactJs
* Redux
* create-react-app
* React Router 4
* redux-form
* React Boostrap Table
* Babel
* Mock API Data: hand rolled
* toastr
* jquery (needed by toastr and bootstrap 4)
* lodash
* Tether
* jquery
